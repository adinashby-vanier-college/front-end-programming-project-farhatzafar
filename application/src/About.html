As a student in the Software Development program at Vanier College, I am the constantly evolving field of computer science. My journey into coding has been both challenging and rewarding, and I often find it comparable to solving a complex puzzle. Each line of code is like a piece that, when carefully placed, contributes to the construction of a larger, more intricate picture.
Throughout my studies, I have had the opportunity to learn several programming languages, including Java, SQL, HTML, CSS, and JavaScript. Among these, Java has particularly captured my interest. I am enthusiastic about the prospect of specializing as a Java developer in the future and am eager to deepen my knowledge in this area.
In addition to coding languages, I have gained valuable experience working with Linux operating systems, particularly the Debian and CentOS distributions.
Looking ahead, I am excited to expand my technical skill set by studying more advanced programming languages, such as C++, C#, and Python. I am also interested in exploring cloud computing platforms, with a particular focus on Microsoft Azure.
